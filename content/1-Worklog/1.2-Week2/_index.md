### Week 2 Objectives:
* Study and practice building a custom Virtual Private Cloud (Amazon VPC with Subnets, Internet Gateway, and Route Tables).
* Learn how to launch and administer Linux-based EC2 instances (Amazon Linux/Ubuntu).
* Experiment with different instance sizes (t3.nano vs t3.micro) to observe resource consumption.
* Install a web server (`httpd`), deploy a sample web page, and configure Security Groups for browser access.
