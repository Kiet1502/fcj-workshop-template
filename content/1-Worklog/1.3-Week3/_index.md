### Week 3 Objectives:
* Launch and administer Windows Server EC2 instances via Remote Desktop (RDP).
* Deploy the **AWS FCJ Management** workshop application onto the server and configure firewalls for stable access.
* Study Amazon S3 object storage (creating buckets, configuring IAM permissions, and Bucket Policies).
* Practice hosting a static web application directly on Amazon S3 using Static Website Hosting.
* Work on-site at the office: Study Amazon RDS relational databases (MySQL), install Docker Engine on EC2, and run sample containers.
* Synthesize weekly evidence, review AWS Billing, and terminate unused cloud resources.
