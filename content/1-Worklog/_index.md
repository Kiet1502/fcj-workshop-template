# WORKLOG OVERVIEW

### Introduction
The Worklog documents the entire learning, practicing, and project development process throughout the **12-week First Cloud Journey (FCAJ)** internship program. The content covers completed AWS hands-on labs, team project development for Aura Academic, on-site office activities, program meetups, and technical documentation drafting.

Through this worklog, you can track the progression of hands-on skills acquired on Amazon Web Services (AWS) alongside professional experience gained in an enterprise environment.

---
